# 動作確認

## 事前準備

```bash
# 設定ファイルのコピー
$ cp .env.example .env

# API_KEYにOpenAIのトークン設定
# TAVILY_API_KEYにTavilyのトークン設定
$ .env
```

.env には以下を設定する。

```
API_KEY=your_openai_api_key
TAVILY_API_KEY=your_tavily_api_key
```

## 実行方法

IDEの実行ボタンを押す

## 確認方法

### chatbot.ipynb

#### 期待する結果

- 猫のキャラクターとして応答すること（語尾に「にゃ」が付く）
- 一般的な質問にはそのまま回答すること
- 検索が必要な質問にはWeb検索を実行し、URL付きで回答すること
